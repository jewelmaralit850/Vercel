RawTXT Maker

Upload or create raw text files: New Upload

Example Python Usage:

import requests
url = "https://rawtxt-maker.vercel.app/api/users"
data = {
  "userId": "user1",
  "filePath": "example.txt",
  "key": "example-91728jekj-91819",
  "newContents": "print('Hello from API')"
}
res = requests.put(url, json=data)
print(res.json())
